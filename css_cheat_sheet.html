background-image

Sirve para establecer una imagen de fondo en la pagina


color

La propiedad de CSS color selecciona el valor de color de primer plano del contenido de elemento de texto y decoraciones de texto.


font

La propiedad de CSS font es una abreviatura de las propiedades de CSS font-style, font-variant, font-weight, font-size, line-height y font-family.


font-family

La propiedad de CSS font-family especifica la fuente de texto que se debe usar para renderizar el texto.


font-style

La propiedad de CSS font-style especifica si el texto debe renderizarse en cursiva o en letra normal.


font-weight

La propiedad de CSS font-weight especifica el grosor de la fuente de texto que se debe usar para renderizar el texto.


width

La propiedad de CSS width especifica el ancho de un elemento. Puede ser un valor absoluto, como 500px, o relativo, como 50%.


height

La propiedad de CSS height especifica lo largo de un elemento.Puede ser un valor absoluto, como 500px, o relativo, como 50%.


list-style

La propiedad de CSS list-style es una abreviatura de las propiedades de CSS list-style-type, list-style-position y list-style-image.

    • margin

    La propiedad de CSS margin especifica el margen exterior de un elemento. Un margen es el espacio entre un elemento y cualquier otro contenido cercano.


    padding

    La propiedad de CSS padding especifica el relleno de un elemento. El relleno es el espacio entre el contenido de un elemento y su borde exterior.


    overflow

    La propiedad de CSS overflow especifica qué sucede si el contenido de un elemento es más grande que el elemento en sí.


    display

    La propiedad de CSS display especifica si un elemento debe ser tratado como un bloque o como un elemento en línea.


    position

    La propiedad de CSS position especifica el tipo de posicionamiento de un elemento. Los elementos posicionados se pueden desplazar, lo que significa que no ocupan espacio en el documento.


    filter

    La propiedad de CSS filter aplica filtros de gráficos en elementos HTML. Los filtros se utilizan para cambiar la apariencia de una imagen (por ejemplo, desenfocarla o cambiar su color).
